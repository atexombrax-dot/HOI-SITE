# HOÏ - Site Officiel

## 🌐 Présentation

Le site officiel HOÏ est la vitrine publique du projet HOÏ.  
Il présente l’écosystème, la vision et les services liés au token HOÏ.

---

## 🎯 Objectif du site

Le site officiel a pour but de :

- Présenter le projet HOÏ au public
- Expliquer la vision et les objectifs
- Donner accès aux ressources principales
- Rediriger vers GitHub et les applications HOÏ
- Faciliter l’adoption de l’écosystème

---

## 🧩 Contenu du site

Le site contient généralement :

- Page d’accueil (présentation du projet)
- Section "À propos de HOÏ"
- Explication du token HOÏ
- Présentation du wallet HOÏ
- Lien vers le whitepaper
- Contact / support

---

## 🔗 Liens importants

- Whitepaper : documentation technique du projet
- GitHub : code source du projet HOÏ
- Wallet HOÏ : accès à l’application (si disponible)
- Dashboard : interface utilisateur

---

## 🚀 Vision

Le site HOÏ est une porte d’entrée vers un écosystème numérique complet basé sur la blockchain, visant à connecter les utilisateurs et structurer un système économique digital moderne.

---
